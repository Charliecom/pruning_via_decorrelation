## Установка зависимостей

Для менеджмента зависимостей в данном репозитории используется утилита uv 

Если у вас нет uv - его можно установить при помощи pipx install uv

Для установки всех необходимых зависимостей используйте команду uv sync

## Overview

Для обучения используется фреймворк Lightning, для логирования MLFlow, для управления конфигом hydra-core

## Задание гиперпараметров через config

Для того, чтобы выставить необходимые гиперпараметры откройте /conf/config.yaml

Поддерживается три датасета: cifar10, cifar100, svhn

А также одна модель: resnet18 

## Запуск обучения

Из корня репозитория выполнить команду uv run nir/train.py

Чтобы поднять mlflow сервер для просмотра логов используйте команду uv run mlflow ui --host 0.0.0.0 --port 5000

## Запуск прунинга

Из корня репозитория выполнить команду uv run nir/prune.py - будет проведен прунинг последнего модели из последнего эксперимента (в config.yaml можно явным образом указать номер run над которым проводить прунинг)

Результаты работы скрипта сохранятся артифакты соответсвующего run
